# OOPMasteryDemo

Demo app to practise C# OOP concepts: encapsulation, inheritance, polymorphism, abstraction, interfaces,
events/delegates, generics, collections, LINQ, and async.

## How to run
1. Open solution in Visual Studio 2022.
2. Build and Run (F5).

## Features
- Accounts demo with Savings and Current account.
- Notification via interface.
- Transaction events and generic repository usage.
